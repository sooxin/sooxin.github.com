我的第二版博客上线啦
发布于2017-08-17，最近更新于2017-08-17

我的第二版博客上线啦！

大约一年半以前，我趁放假在家的机会做了我的第一版博客，托管在 GitHub 上面，但是当时更多是为了尝试和学习 git 和 Amaze UI 而用来练手的，做的也简陋，加之当时技术有限和后来的一阵繁忙，于是在短暂的更新一段时间后，就停手做其他的去了。不过我仍保持没事写点东西的习惯，只是大多记录在自己的云笔记里了，偶尔放上一两篇到简书上。

转眼大三就结束了。这个暑假里除了和同学做项目，也寻思着给自己的些许文章找个窝。于是再次想到在 GitHub 上重建自己的博客。两年的学习也让我眼界和想法开阔了许多，从一开始想用 hexo 或者 Jekyll 来建站到最后想“为什么不自己写一个类似的静态博客生成器呢”。于是说干就干，用我最熟悉的Java写了这个名为 soobo 的静态博客生成器。不过目前 soobo 这个项目只完工了一半不到，还有许多需要我去学习然后完善的地方。不过我迫不及待的想要先把我的博客建立起来，于是借着这个半成品的 soobo 和我的手工更改，先让我的博客上线再说。

于是我的第二版博客上线了，以后我会把以前的文章慢慢搬过来。

以前学习 SpringMVC 的时候，我发现做笔记真的很容易让我进入学习状态，加之学会了 markdown 这个利器，让我竟有些享受敲键盘写作的感觉了。

哈哈，学不可以已，“笔”也不能缀！

作为第二版博客的第一篇，想到了哪儿写到哪，有些絮叨了，先就此打住吧！
